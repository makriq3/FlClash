# v0.8.95

## Summary

This release turns the fork into a more self-contained Android product.

## User-facing changes

- Added in-app Android self-update support.
- Switched the app to standalone product identifiers and package names.

## Technical notes

- Android package names, service bindings, and release assets were moved to the fork namespace.
- The updater flow was added together with the provider paths it needs for installation.
