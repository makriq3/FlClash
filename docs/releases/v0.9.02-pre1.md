# v0.9.02-pre1

## Для кого этот релиз

Предрелиз исправляет первые регрессии ветки `v0.9.02` перед дальнейшей обкаткой.

## Что изменилось для пользователя

- Восстановлена корректная работа Android split tunneling для списков приложений из файлов и URL.
- Убраны ложные совпадения приложений, если список установленных пакетов временно недоступен.
- Сохранены route-настройки из активного профиля без лишней runtime-перезаписи.

## Технические заметки

- Runtime builder снова поднимает package-list поля из исходного profile YAML до этапа нормализации.
- Внешние exact-селекторы больше не считаются надёжными без инвентаризации установленных приложений.
- Поля маршрутизации из профиля остаются источником истины во время Android runtime patching.
