# v0.9.02-pre1

## Summary

This bugfix pre-release repairs the first cut of the `v0.9.02` line.

## User-facing changes

- Restored reliable handling of profile-based package lists from files and URLs.
- Prevented unrelated app matches when installed-app inventory is temporarily unavailable.
- Kept route settings from the active profile intact.

## Technical notes

- The runtime builder now restores package-list fields from raw profile YAML before normalization.
- External exact selectors are no longer trusted without installed-app inventory.
- Profile-defined routing fields now remain the source of truth during Android runtime patching.
