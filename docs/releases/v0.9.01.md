# v0.9.01

## Summary

This stable release completes the first full Android split tunneling line for the fork.

## User-facing changes

- Android split tunneling can now be controlled directly from profile YAML.
- App rules support files, URLs, masks, regex selectors, and exclusions.
- The Android updater and supporting flows are more reliable.

## Technical notes

- Profile data now drives Android app-rule generation end to end.
- File handling and provider boundaries were tightened around split tunneling assets.
- The release also consolidates the access-flow and updater fixes from the pre-release series.
