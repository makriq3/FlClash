# v0.8.93

## Summary

This release starts the independent fork line and introduces the first Android privacy hardening changes.

## User-facing changes

- Android VPN mode no longer exposes the usual local proxy listeners.
- The fork can now be released and maintained from its own repository.
- Public project docs were updated for the standalone product line.

## Technical notes

- The Android runtime hardening layer forces local listener ports closed in VPN mode.
- The hardened path also disables `external-controller` and keeps `allow-lan` off.
- CI and release workflows were adapted to build and publish from the fork repository.
