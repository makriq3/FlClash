# v0.9.01-pre3

## Summary

This pre-release adds remote package lists and makes the active profile the source of truth.

## User-facing changes

- Added URL-backed package lists for Android split tunneling.
- Improved consistency between the active profile and the applied app rules.

## Technical notes

- Remote list sources are resolved before Android app rules are generated.
- The active profile now wins over stale app-side state during split tunneling setup.
